## LiveSplit
 
Download and install [LiveSplit](https://livesplit.org/downloads/).
 
Add the splits:
 
1. Right-click
2. Open Splits
3. From File...
4. Splits.lss
 
Add the auto splitter:
 
1. Right-click
2. Edit Layout...
3. `+`
4. Control
5. Scriptable Auto Splitter
6. Double-click on the new entry
7. Browser...
8. Autosplit.asl
9. OK
10. OK
 
I used [Cheat Engine](https://www.cheatengine.org/) to find the pointer paths.
 
## OBS
 
Download and install [OBS](https://obsproject.com/download)
 
Add a digital clock:
 
1. Download `datetime.lua` from [here](https://obsproject.com/forum/resources/datetime-digital-clock.882/).
2. Add Source > Text
3. Name it `Text01` and click `OK`
4. Select font > Size: 24
5. OK
6. OK
7. Tools > Scripts
8. `+`
9. Select `datetime.lua`
10. Datetime format: %Y-%m-%d %H:%M:%S %z
11. Text Source: Text01`
12. Close
